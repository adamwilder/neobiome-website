Quick option — create with Canva (free):

  2. Background fill: #F6F4EF
  3. Add text "NEOBIOME" — Playfair Display Bold, ~100pt
     Color the N-E-O letters #3C5E3C and B-I-O-M-E letters #BB8F4E

Quick option — create with Figma, Adobe Express, or any image editor.

Once the file exists at /assets/og-image.png, LinkedIn and other
social platforms will pick it up automatically via the og:image
